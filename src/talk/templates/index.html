Start your fitness journey.

Welcome to TALK Fitness! This web app was built with the Python framework Flask, and features a SQLite database for secure user login and registration, as well as user data storage/access. The frontend was built with Bootstrap 5.0.

Once an account is created, users can import their fitness logs for various athletic activities through Strava or by manually entering their stats. Challenges can be completed which earn extra points, moving you up the leaderboard ranks!
