TALK Fitness

web application that will take user fitness data, either manually entered or entered through an existing fitness tracker’s API, and log it into a user account. Based on their input, we will create a leaderboard showing their username and their fitness score. Login authentication will hopefully use Google’s OAuth. The user will be able to record their information and see other users’ fitness goals and fitness history.
